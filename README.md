# Python Tic-Tac-Toe CLI

A command-line version of the classic Tic-Tac-Toe game, playing against a computer opponent.

## Key Features
- **2D List Data Structure:** The board is managed as a matrix [3][3].
- **Input Validation:** Prevents overwriting moves or entering invalid inputs.
- **Game Logic:** Automatically detects win conditions (Rows, Columns, Diagonals) and draws.
